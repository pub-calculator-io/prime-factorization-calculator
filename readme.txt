The prime factorization calculator finds a number's prime factors. The calculator demonstrates the prime factors tree and all factors of the number. 

[https://www.calculator.io/prime-factorization-calculator/](https://www.calculator.io/prime-factorization-calculator/)

== Usage ==

Add the Prime Factorization Calculator shortcode to your page, post or sidebar:

`[ci_prime_factorization_calculator]`

Add the following code to your template where you would like the Prime Factorization Calculator to appear:

`<?php display_ci_prime_factorization_calculator(); ?>`

== Installation ==

1. Upload the Prime Factorization Calculator /ci_prime_factorization_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Prime Factorization Calculator plugin through the "Plugins" menu in WordPress.
